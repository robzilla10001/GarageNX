FONTS

Bundled with the repository:
  DejaVuSansMono.ttf  — used for the hex viewer (fixed-width alignment).
                        Bitstream Vera / DejaVu license (permissive, freely
                        redistributable). Do not remove; the hex view needs it.

Must be added manually before building (NOT bundled — see below):
  Inter-Regular.ttf
  Inter-Bold.ttf

  Download Inter from: https://rsms.me/inter/
  Open the zip, go into "Inter Desktop/", and copy those two TTF files here.
  Inter is SIL OFL 1.1 licensed and free to redistribute; it is simply not
  vendored in this repo to keep it lean. The UI will not render text without it.
